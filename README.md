<h1 align="center">Text-Editor</h1>
<h2 align="center">Название и цель лабораторной работы</h2>
<p style="font-size: 18; font-weight: bold;">Лабораторная работа 1. Разработка пользовательского интерфейса (GUI) для языкового процессора</p>

<h4>Цель работы:</h4>
<p>Создание кроссплатформенного графического интерфейса (GUI) для языкового процессора в виде специализированного текстового редактора</p>
<h2 align="center">Сведения об авторе</h2>
<p>Работу выполнил MaKiToShI</p>

<h2 align="center">Описание проекта</h2>
Текстовый редактор — это приложение с графическим интерфейсом, разработанное на Python с использованием библиотеки PyQt6. Приложение представляет собой специализированный текстовый редактор, который в дальнейшем будет дополнен функциями синтаксического анализатора.

<h2 align="center">Используемые технологии</h2>
<h4>Язык программирования:</h4>
<p>Python 3.12 - интерпретируемый, высокоуровневый язык программирования</h4>

<h4>Фреймворк для GUI</h4>
<ul>
  <li>PyQt6 - библиотека для создания графического интерфейса на Python</li>
  <li>Qt Designer - визуальный редактор для проектирования интерфейса (.ui файлы)</li>
</ul>

<h4>Среда разработки:</h4>
<p>VS Code (Visual Studio Code) - мощный редактор кода</p>

<h4>Дополнительные инструменты:</h4>
<p>PyQt6.uic - модуль для загрузки .ui файлов в Python</p>
<p>Git - система контроля версий</p>

# Руководство пользователя

## Интерфейс программы
Главное окно текстового редактора состоит из следующих элементов: 
1. **Заголовок окна** — отображает название программы и имя текущего файла
2. **Главное меню** — содержит все доступные команды
3. **Панель инструментов** — кнопки быстрого доступа
4. **Область редактирования** — текстовое поле для ввода и редактирования текста
5. **Область вывода результатов** — область для отображения результатов работы анализатора
6. **Строка состояния** — отображает информацию о состоянии работы приложения

## Меню «Пуск»
> [!NOTE]
> **`Находится в разработке, но предполагает следующее:`**

**Запуск анализатора** (`F5`) — запускает синтаксический анализ текста из области редактирования.

**Результаты анализа:**
1. Ошибочные строки отмечаются красным цветом с указанием позиции ошибки
2. При щелчке на сообщении об ошибке курсор переходит к ошибочному фрагменту

## Работа с областями редактирования и вывода

> [!NOTE]
> **Область редактирования:** предназначена для ввода и редактирования текста. Поддерживаются все стандартные операции редактирования.
> 
> **Область вывода результатов:** отображает результаты работы синтаксического анализатора. Область доступна только для чтения.
> 
> **Изменение размеров областей:** перетаскивайте разделитель между областями мышью.
> 
> **Изменение размера шрифта в области для редактирования:** при помощи колёсика мышки можно регулировать масштаб текста.

**Для увеличения масштаба** зажмите клавишу `Ctrl` и прокрутите колёсико мыши вперёд (от себя) - размер текста будет плавно увеличиваться:

<img src="https://github.com/MaKiToShI21/Text-Editor/blob/main/images/ru/approximation.png" width="450">

**Для уменьшения масштаба** зажмите клавишу `Ctrl` и прокрутите колёсико мыши назад (на себя) - размер текста будет плавно уменьшаться:

<img src="https://github.com/MaKiToShI21/Text-Editor/blob/main/images/ru/distancing.png" width="450">

## Подсветка синтаксиса
Редактор поддерживает подсветку синтаксиса для языка `Python`:
1. Ключевые слова (if, else, def, class, import и др.)
2. Строки и числа
3. Комментарии
4. Функции и методы
5. Встроенные исключения и декораторы

<img src="https://github.com/MaKiToShI21/Text-Editor/blob/main/images/ru/syntaxis.png" width="450">
